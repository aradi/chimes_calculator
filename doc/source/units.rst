ChIMES Units
=====================


ChIMES uses the following base units:

=========  ========
Property   Unit
=========  ========
Distance   Angstroms ("Ang") 
Energy     kcal mol :math:`^{-1}`
Stress     kcal mol :math:`^{-1}` Ang :math:`^{-3}`
Force      kcal mol :math:`^{-1}` Ang :math:`^{-1}`
=========  ========
